docker run

docker run | Docker Documentation

新建容器并启动

参数 说明
–name=”Name” 容器名字，区分容器
-d 后台方式运行
-it 使用交互方式运行，进入容器查看内容
-p 指定容器端口 -p 8080:8080
-P 随机指定端口

docker ps

docker ps | Docker Documentation

列出容器

参数 描述
-a 列出当前正在运行的容器+历史运行过
-n=? 列出最近创建的n个容器
-q 只显示容器ID
