DockerFile

Dockerfile就是用来创建docker镜像的构建文件——命令脚本。

案例 CentOS

Docker指令

基础知识：

  1. 每个保留关键字（指令）都必须是大写字母
  2. 从上到下顺序执行
  3. #表示注释
  4. 每一个指令都会创建一个新的镜像层，并提交！
命令 解释
FROM 基础镜像
MAINTAINER 镜像编写者，姓名+邮箱
RUN 运行
ADD 添加
WORKDIR 镜像工作目录
VOLUME 挂载的目录
EXPOSE 暴露端口
CMD 指定容器启动时要运行的命令，最后一个生效
ENTRYPOINT 指定容器启动时要运行的命令
ONBUILD 被继承时运行该指令，触发指令
COPY 类似ADD，将文件拷贝到镜像中
ENV 构建时设置环境变量
