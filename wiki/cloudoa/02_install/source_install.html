服务器部署

服务器源码为server文件，使用IDEA工具编写，还请使用IDEA打开项目，包管理工具为Maven。

配置文件

打开servuce-oa/src/main/resources/application-dev.yaml，修改配置信息

  • app-id：微信公众号的appId
  • secret：微信公众号的secret
  • prefix：微信公众号的授权回调页面域名

👈获取方式：微信公众号测试申请
